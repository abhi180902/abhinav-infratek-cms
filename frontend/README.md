# Abhinav Infratek CMS

A full-stack Content Management System (CMS) for managing the Abhinav Infratek construction company website.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- React
- Vite

## Setup

Create the required environment variables before running the application.

Frontend:

```bash
npm install
npm run dev
```

Backend:

```bash
mvn spring-boot:run
```
